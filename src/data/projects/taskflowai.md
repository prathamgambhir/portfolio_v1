# TaskFlow | AI Task Manager

## Overview

TaskFlow is a modern, responsive SaaS landing page featuring smooth animations and a polished dark/light themed design with purple accents built with Next.js, Tailwind CSS, and Framer Motion. 

The focus was on **UI/UX excellence**, **performance optimization**, and **modern frontend development practices**.

---

## What Users Can Do

- **Experience Smooth Animations**  
  Enjoy fluid animations and interactions powered by Framer Motion throughout the entire page.

- **Toggle Themes**  
  Switch between dark and light modes with a seamless transition experience.

- **View Responsive Design**  
  Experience a fully responsive layout that adapts perfectly to all screen sizes.

- **Navigate Effortlessly**  
  Use the intuitive navbar with mobile menu support to explore different sections.

- **Learn About Product**  
  Access comprehensive information about features, pricing, and FAQs.

---

## Why I Built This

I wanted to build a **production-ready SaaS landing page** to practice and demonstrate:

- Modern UI/UX design principles  
- Smooth, performance-optimized animations  
- Clean component architecture  
- Responsive web development  
- Dark mode implementation  
- Component composition patterns  

TaskFlow was built to showcase **frontend development expertise** and serve as a foundation for future SaaS products.

---

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Heroicons
- **UI Components:** Shadcn
- **Theme Management:** next-themes
- **Utilities:** clsx, tailwind-merge, class-variance-authority

---

## Key Engineering Highlights

- Implemented **Framer Motion** animations with staggered reveal effects and drag gestures
- Built **theme provider** with next-themes for seamless dark/light mode switching
- Created **reusable component library** including feature cards, buttons, and animated sections
- Designed **responsive grid layouts** that adapt from mobile to desktop
- Added **interactive elements** including draggable cursor and message animations
- Implemented **scroll-triggered animations** using viewport detection
- Built **accessible FAQ section** with smooth expand/collapse animations
- Created **logo ticker** with smooth scroll for social proof and credibility
